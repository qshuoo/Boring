# 一只弱鸡贪吃蛇    

## step1 利用JFrame创建游戏窗口     
继承Swing的JFrame类设计窗口的尺寸位置等属性
## step2 用JPanel设计窗口布局      
继承Swing的JPanel设计窗口的布局，之后的游戏元素均在JPanel中放置	
## step3 编写基本组成元素Point   
游戏中的基本元素就时一个一个点    蛇和食物 以及 整个布局 都抽象为一个一个点，点的属性为X,Y表示其坐标
## step4 设计蛇 并在窗口中显示蛇
蛇的基本属性有蛇自身，这里用一个LinkedList<Point>表示它是一个由点构成的链表，选用LinkedList的原因是对头和尾的处理更加简单，另一个属性是方向，这里用一个点表示，这个点的格式固定为（x,0）或（0，y）其中x,y的值为窗口布局中选择的一个点的尺寸大小的正负值，蛇头坐标的Point加上方向的Point值就是蛇头的下一位置，蛇的基本方法有移动，吃食物和获取蛇头的下一个点，获取下一个点的目的时辅助判断吃食物
## step5 让蛇动起来 
在蛇的移动方法中，这里设计的是通过蛇链表的增加和删除实现的，第一步获得蛇头节点的下一个位置P，第二步在蛇链表的头部节点增加下一个位置P，第三步删除蛇链表的尾部节点
## step6 控制蛇的移动
设置监听按键响应，改变蛇的方向，当按下方向键时蛇的方向改为对应的方向，但是不能向相反的方向改变，当按下相反的方向时，方向不做改变
## setp7 生成食物
食物类的属性即是一个点，用Point类表示，在其初始化的时候会给Point在范围内随机赋值，包含的方法只有重置位置，表示其在被蛇吃之后位置在范围内重置,也就是表示食物类在游戏运行的期间不会消失，而是用重置其位置表示其被吃的行为
